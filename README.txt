PcaptoLVX2 - PCAP/PCAPNG to LVX2 Converter
===========================================

简介
----
PcaptoLVX2 是一个用于将 PCAP/PCAPNG 文件转换为 LVX2 格式的工具。该工具支持中文路径，并提供了图形界面来选择文件。

系统要求
--------
- Windows 10 或更高版本
- Wireshark
- Visual C++ 2022 Runtime

安装步骤
--------
1. 运行 PcaptoLVX2Setup.msi
2. 按照安装向导的提示完成安装
3. 如果系统提示缺少依赖项，请安装：
   - Visual C++ 2022 Runtime
   - Wireshark（如果尚未安装）

使用方法
--------
方法一：通过文件选择对话框
1. 双击运行 PcaptoLVX2.exe
2. 在弹出的文件选择对话框中选择要转换的 PCAP/PCAPNG 文件
3. 程序会自动在相同目录下生成同名的 LVX2 文件

方法二：通过命令行
1. 打开命令提示符
2. 运行命令：PcaptoLVX2.exe <输入文件路径>
   例如：PcaptoLVX2.exe D:\test.pcapng

注意事项
--------
1. 确保有足够的磁盘空间用于文件转换
2. 转换过程中请勿关闭程序
3. 支持中文路径和文件名
4. 如果遇到问题，请检查：
   - Wireshark 是否正确安装
   - 输入文件是否存在且可访问
   - 输出目录是否有写入权限

常见问题
--------
Q: 程序无法启动
A: 请确保已安装 Visual C++ 2022 Runtime 和 Wireshark

Q: 转换失败
A: 请检查输入文件格式是否正确，以及是否有足够的磁盘空间

Q: 找不到输出文件
A: 输出文件将保存在与输入文件相同的目录下，文件名相同但扩展名为 .lvx2
